## LAZYDOCKER AUTO-INSTALL
![lazydocker](img/lazydocker.png)

Script for install lazydocker - x86-arch64 and ARM v7 versions

Install x86 x64:
```
curl -fsSL https://raw.githubusercontent.com/koko004/lazydocker-autoinstall/refs/heads/main/install-lazydocker-x86-64.sh | bash
```

Install Arm v7:
```
curl -fsSL https://raw.githubusercontent.com/koko004/lazydocker-autoinstall/refs/heads/main/install-lazydocker-armv7.sh | bash
```
